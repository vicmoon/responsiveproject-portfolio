JavaScript Digital Clock

Working on this project was so much fun ! If a few months ago I would be really concerned whenever a bug comes up or I get stuck , now I know that most bugs have a solution or a reason, it's just the mind which needs a break. The date and time on the internet are certainly looking better with Moment.js. Although modern applications require libraries which take less space, Moment.js is still being downloaded at an astonishing 12 million times per week!

React Weather App

Why another weather app? Well, because using the Open Weather API makes for a perfect learning experience. This one has a fresh interface with vibrant colours and a style inspired by airports. I like the frenetic movement, the sounds, the voices, the timetables. This time I've learned how to use React - a Javascript library for building user interfaces. It has been very interesting to learn about components, external components, Ajax, node.js, props and States. The web is slowly but surely becoming clearer to me.
